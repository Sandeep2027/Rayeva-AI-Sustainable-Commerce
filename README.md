# Rayeva AI – Sustainable Commerce System
**Full Stack / AI Intern Assignment**  
**Location:** Kadapa, Andhra Pradesh

## Objective
AI-powered modules that reduce manual catalog effort, improve B2B proposals, automate impact reporting, and simulate WhatsApp support.

## Modules Implemented
- **Module 1**: AI Auto-Category & Tag Generator (TF-IDF + Logistic Regression)
- **Module 2**: AI B2B Proposal Generator
- **Module 3**: AI Impact Reporting Generator
- **Module 4**: WhatsApp Support Bot (Simulation with 10 FAQ + Live Chat)

## Tech Stack
- FastAPI + Uvicorn
- scikit-learn (ML)
- NLTK + VADER Sentiment
- SQLite
- Tailwind CSS + Chart.js

## How to Run
```bash
pip install fastapi uvicorn scikit-learn nltk
uvicorn main:app --reload
